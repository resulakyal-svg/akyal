# GANO Hesaplama Uygulaması (.NET MAUI)

Bu proje öğrencilerin vize ve final notlarına göre otomatik GANO hesaplayan bir mobil/masaüstü uygulamadır.

## Özellikler
- Ders ekleme
- Ders silme
- Vize ve Final notu girme
- Otomatik ortalama hesaplama (Vize %40 - Final %60)
- Harf notu üretme
- AKTS ağırlıklı GANO hesaplama
- Ortalamaya dahil / hariç ders seçimi
- SQLite veritabanı kullanımı

## Teknolojiler
- C#
- .NET MAUI
- SQLite

## Hesaplama Formülü
Ortalama = Vize * 0.4 + Final * 0.6

GANO = AKTS ağırlıklı ortalama
